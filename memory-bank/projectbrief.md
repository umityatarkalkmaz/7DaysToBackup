# Project Brief - 7DaysToBackup

## Proje Özeti
7 Days to Die oyunu için save dosyalarını yönetmeye yarayan bir masaüstü uygulaması.

## Temel Gereksinimler
- Save dosyalarını yedekleme (backup)
- Save dosyalarını silme (delete)
- Save dosyalarını dışa aktarma (export - zip formatında)
- Save dosyalarını içe aktarma (import - zip'ten)
- Çoklu dil desteği (Türkçe, İngilizce)
- Koyu tema arayüz
- Çapraz platform desteği (Windows, macOS, Linux)

## Hedef Kullanıcılar
- 7 Days to Die oyuncuları
- Save dosyalarını korumak isteyen kullanıcılar
- Farklı save dosyalarını yönetmek isteyen oyuncular

## Temel Hedefler
1. Kullanımı kolay ve sezgisel arayüz
2. Güvenli yedekleme işlemleri
3. Tüm işletim sistemlerinde sorunsuz çalışma
4. Türkçe ve İngilizce dil desteği
